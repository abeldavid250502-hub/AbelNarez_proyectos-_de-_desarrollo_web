# Proyectos de Desarrollo Web

Este repositorio contiene una colección de mis proyectos de desarrollo web realizados con fines de práctica, aprendizaje y uso académico.

Los proyectos incluidos en este repositorio fueron desarrollados utilizando tecnologías como:

- **PHP**
- **HTML**
- **CSS**
- **SQL**

##  Acerca de este repositorio

Este repositorio fue creado para organizar y mostrar diferentes proyectos de desarrollo web en los que he trabajado durante mi proceso de formación como estudiante de Desarrollo de Software.

El objetivo principal de este repositorio es fortalecer mis habilidades en:

- Desarrollo frontend
- Desarrollo backend
- Manejo de bases de datos
- Resolución de problemas
- Creación de aplicaciones web funcionales

##  Tecnologías utilizadas

- PHP
- HTML5
- CSS3
- SQL
- MySQL / fundamentos de bases de datos

## Proyectos incluidos

Actualmente este repositorio contiene **3 proyectos**, cada uno enfocado en diferentes áreas del desarrollo web.

### 1. Arca Animal (Gestion de una veterinaria)
Proyecto escolar en donde trabajamos gestionando por medio de modulos de CRUD , en donde utilize las herramientas para front-end HTML,CSS y para el back-end PHP y SQL, asi mismo para metodologia del Software utlizamos el Proceso Unificado Agil (AUP) 

### 2. Aceites y Lubricantes (Aplicacion Web )
Proyecto escolar en donde levantamos requerimientos de una micro-empresa cerca de la zona escolar , donde el cliente nos solicito una pagina web (no esta en linea) trabajamos solo con herramientas de front-end HTMS y CSS . donde utilizamos la metodoligia SCRUM proyecto en el cual me toco ser SCRUM MASTER  


### 3. Registro 
Proyecto escolar en donde solo nos solicitaron aprender a trabajar con las herramientas HTML y un poco de CSS ya que nos enfocamos mas en PHP y SQLlite para la base de datos
